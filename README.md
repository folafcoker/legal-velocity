# Legal Velocity — Elaine Approval Turnaround

Dashboard for tracking how long contracts spend with Elaine during the internal approval loop.

## What it does

Single-page HTML file. No build step, no server needed — just open in a browser.

Shows every contract that has been sent to Elaine for approval, grouped by contract. Click any row to expand and see every individual turn: when it was sent to Elaine, when it came back, and how many days it took.

## How to open

```
open legal-velocity.html
```

Or just double-click `legal-velocity.html` in Finder.

## How to share with the COO

Drop the HTML file into a shared Google Drive / Notion attachment / email. It's fully self-contained — one file, no dependencies.
